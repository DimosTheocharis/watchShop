# How to run

## 1. Git clone the project

```
  git clone https://github.com/DimosTheocharis/watchShop.git .
```

## 2. Open the cloned project

## 3. Run the app with flask

```
  python app.py
```

## 4. Open the app

Head to _http://127.0.0.1:5000/_
